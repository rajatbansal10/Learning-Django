## Django Project

#### Create Virtual Environment 
`python -m venv my_env1`

#### Install Django
`pip install Django==5.2`

#### Create Django Project
`django-admin startproject myFirstProject`

#### Run Server
`python manage.py runserver`

## Django Architecture
Django works on MVT architecture.

1. M - Model - models.py - DB 
2. V - View - views.py - Business Logic
3. T - Template - HTML Files - Presentation Logic
